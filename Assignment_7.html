  1. Create a box plot figure that shows the variation in heights among Star Wars species. Do the values in your summary table line up with the plot? Code so that each box plot (Species) is be filled with a different color. Change your axes labels so that species is capitalized and height is capitalized and shows units (cm). Use the theme_minimal option. Then look into the annotate or geom_text functions and add the text “p=1.2E-0.5” to the top right corner.
p1 <- ggplot(data = sw_clean, aes(x = species, y = height, fill = species)) + 
  geom_boxplot() +
  labs(x="Species",
       y="Height (cm)") +
  theme_minimal() +
  geom_text(aes(x = NA, y = max(height), label = "p = 1.2E-0.5"), size = 3.2) +
  theme_classic()

print(p1)
